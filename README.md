# Flow
An open source split mechanical keyboard powered by [Ergogen](https://github.com/ergogen/ergogen) and kicad


![Main layout](flow.png)

# Features (So far)
- Hotswappable with mx-style switches
- Flippable PCB
- Stabilizers
- Thumb Cluster with 3 keys (on each hand)
